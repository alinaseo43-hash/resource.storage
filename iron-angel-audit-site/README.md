# Iron Angel Audit Site

Окремий статичний сайт для презентації аналізу `iron-angel.ua`.

## Структура

- `index.html` - розмітка сайту.
- `styles.css` - незалежні стилі у візуальному стилі поточного проєкту.
- `script.js` - рендер етапів, навігація та фонова анімація.
- `assets/screenshots/` - папка для скринів, які будуть додані другим етапом.
